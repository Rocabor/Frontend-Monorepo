### What I learned

In this project, I reinforced my skills with:

- **Advanced CSS Positioning**: I used absolute positioning to overlay the remaining storage indicator and the decorative triangle.
- **Customizing the `<progress>` element**: I learned to fully style the native progress bar.
- **Tailwind CSS v4 with `@theme` and `@layer`**: Custom color configuration and utilities.
- **Responsive background image**: Switching between bg-mobile and bg-desktop using Tailwind variants.


### Continued development

In future projects, I want to continue improving in:

- Mastering Tailwind CSS v4 and its new features
- Implementing more complex patterns with CSS Grid
- Better accessibility practices (ARIA labels, landmarks)
- Image and resource optimization
